# Win10ChatApp
A desktop chat application compiled for Windows 10 that uses C# WinForms for the client and C++ for the server.

## Editting the client
```
cd BarebonesServer
make
```

## Editting the server
The server should be compiled in visual studio 2019

## Start the server
```
./startServer.bat
```

## Start the client
```
./startClient.bat
```
